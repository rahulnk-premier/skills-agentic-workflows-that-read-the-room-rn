## GitHub Changelog — July 2026

- **[Copilot CLI no longer needs a PAT in GitHub Actions](https://github.blog/changelog/2026-07-02-copilot-cli-no-longer-needs-a-personal-access-token-in-github-actions)** — You can now use `GITHUB_TOKEN` to run Copilot CLI in Actions workflows. No more storing personal access tokens as secrets. *(GitHub Changelog)*

- **[Copilot agent session streaming in public preview](https://github.blog/changelog/2026-07-02-copilot-agent-session-streaming-is-now-in-public-preview)** — Enterprise Cloud customers can now stream Copilot agent session data across all Copilot clients. *(GitHub Changelog)*

- **[Issue fields are now generally available](https://github.blog/changelog/2026-07-02-issue-fields-are-now-generally-available)** — Custom issue fields are GA for all GitHub organizations and coming to GitHub Enterprise Server 3.23. *(GitHub Changelog)*

- **[GitHub Models fully retires July 30, 2026](https://github.blog/changelog/2026-07-01-github-models-is-being-fully-retired-on-july-30-2026)** — If you use GitHub Models, migrate before July 30th. *(GitHub Changelog)*

- **[Enterprise managed-settings.json is generally available](https://github.blog/changelog/2026-07-01-enterprise-managed-settings-json-is-generally-available)** — Enterprise Cloud customers can govern Copilot AI standards through a `managed-settings.json` file in a `.github-private` repository. *(GitHub Changelog)*
